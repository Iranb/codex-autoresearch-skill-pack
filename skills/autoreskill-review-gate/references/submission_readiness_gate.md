# Submission Readiness Gate

Required:

- `paper/main.tex`
- `paper/main.pdf`
- citation integrity verified
- high/critical issues closed or claims downgraded
- `paper/VENUE_PROFILE.md`
- target-venue summary
- venue-required significance, ethics, limitations, reproducibility, cover-letter, or checklist artifacts
- reporting/checklist gap list
- data/code availability notes

The final package is venue-profile driven. `submission_ready.json.required_artifacts` is the authority for venue-specific files; NMI is one profile, not the only target.
