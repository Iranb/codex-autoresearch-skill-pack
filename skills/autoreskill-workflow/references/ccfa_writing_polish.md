# CCF-A Writing Polish Router

Use this file as the backward-compatible entry point for CCF-A/top-tier English writing and polishing requests. Do not treat this file as the full checklist. Route to the right layer below.

## Three-Layer Design

1. **Corpus Evidence Layer**: read `ccfa_writing_style_corpus_audit.md` when collecting or validating writing-style evidence from papers.cool, official venue pages, proceedings, OpenReview, arXiv, PaperNexus, or local PDFs.
2. **Writing Principle Layer**: read `ccfa_writing_principles.md` when the user asks for writing advice, CCF-A style checks, abstract/introduction/Figure 1 guidance, contribution framing, claim calibration, or a reusable checklist.
3. **Manuscript Application Layer**: read `ccfa_manuscript_revision_workflow.md` when revising, polishing, or reviewing a concrete manuscript. This layer must produce `.autoreskill/paper/CCFA_WRITING_AUDIT.md` before sentence-level rewriting.

## Routing Rules

- If the user asks "how should I write?" or "what should I check?", read `ccfa_writing_principles.md`.
- If the user asks to analyze oral/best/award papers, harvest papers.cool, verify labels, or justify the advice from a corpus, read `ccfa_writing_style_corpus_audit.md`.
- If the user provides a manuscript or asks to polish/revise a paper, read both `ccfa_writing_principles.md` and `ccfa_manuscript_revision_workflow.md`; also read `ccfa_writing_style_corpus_audit.md` if corpus-derived findings are cited.
- If the user asks for wording to be less defensive, stronger, more direct, less caveated, less apologetic, or less self-undermining, read `ccfa_writing_principles.md` and apply the non-defensive writing posture. For a concrete manuscript, also read `ccfa_manuscript_revision_workflow.md` and record the pass in `CCFA_WRITING_AUDIT.md`.

## Non-Negotiable Editing Order

For concrete manuscripts:

1. Audit the story spine and claim scope.
2. Write or update `.autoreskill/paper/CCFA_WRITING_AUDIT.md`.
3. Repair title, abstract, introduction, Figure 1, method motivation, experiment organization, related work positioning, and claim wording.
4. Run the non-defensive writing pass after claim calibration.
5. Polish paragraph flow and sentence-level English last.

Do not perform only grammar cleanup or vocabulary substitution and call it CCF-A polishing.
